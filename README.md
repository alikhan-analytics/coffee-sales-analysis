# coffee-sales-analysis
Coffee Shop Sales Dashboard showcasing interactive analysis of sales, footfall, product categories, order sizes, and store performance. Visualizes peak hours, top products, and weekday trends using charts and graphs. Built with Excel for data analysis and visualization.
